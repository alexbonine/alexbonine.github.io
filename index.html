UI Components Library and Table

ToutApp

UI Components Library and Table

Created UI components library to give our apps common design language, interactions, and styling documented in live coding environment. Developer tools were created to increase productivity.

UI components library begun with Gmail Chrome extension to prove out architecture before moving to OWA add-in and into the main web app. A new package now defines the grid used throughout the application. The grid is completely customizable and we are investigating the possibility of open-sourcing.

Two major goals of refactoring ToutApp's front-end codebase were code reuse and creating a consistent style. With the launch of my first project, I begun the UI component library. The library contains all the styles and shared React components across ToutApp's clients from Gmail and OWA to our main web and even integrating into Salesforce and our desktop Outlook client. To increase developer productivity, tools and a playground were created. Working with our designer, we have developed living documentation to allow developers and designers to play, re-style, and learn to use the library.

Recently, I tackled a new challenge of creating a common grid to be used throughout ToutApp. The solution is flexbox-based and includes features such as sorting, searching across multiple columns, pagination, and filters with data either pre-fetched or hitting an API. We are investigating open-sourcing the project.

Skills: React, ES6, Redux, UI/UX Design, HTML, Sass, Storybook, Webpack, NPM, Git

Gmail & Outlook OWA Extensions

ToutApp

Gmail & Outlook OWA Extensions

Wrote Chrome Extension in React which saw 1.9 million emails sent from Gmail, a 100% increase, within three months.

Proved out common components architecture by creating Outlook Web add-in.

The Gmail Chrome extension rewrite was my first project undertaken while at ToutApp. I convinced our CEO to use the project as a proof of concept for a new front-end architecture. After researching and protyping, I choose React with Redux. I had used React on side projects before and liked it's one-way data flow and component nature. jQuery was used to inject the React directly into the Gmail DOM. A UI component library was started to share components across apps. When the extension was released in December 2015, the old version saw one million emails a month. By March 2016, 1.9 million emails were sent.

Following the success of the Chrome extension, we decided to see how well the architecture and UI component library sped up the development cycle by creating an Outlook web app (OWA) add-in. When finished, the add-in helped customers use ToutApp from their online Outlook mailbox.
